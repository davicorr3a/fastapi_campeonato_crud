Sistema de Gerenciamento de Campeonato de Futebol

Este sistema foi desenvolvido como parte do trabalho prático da disciplina de Banco de Dados da UFS. Ele permite gerenciar clubes, estádios, partidas e muito mais, utilizando abordagens com banco de dados relacional (PostgreSQL) e não relacional (MongoDB).

Banco Relacional: Gerencie os dados com estrutura formal, integridade referencial e relacionamentos complexos.

Banco Não Relacional: Experimente flexibilidade e agilidade na manipulação dos dados com MongoDB.
